项目经验

  • 路灯配电与单灯管控智能化关键技术的研究 硕士课题/公司项目 2024.1—至今

    • 课题描述：针对城市箱变用测量仪表无法较为全面地监测城市箱变的电参量及开关状态量、接地故障无法及时精准识别导致灯杆人身触电的问题，本课题设计了一种单相柔性智能配电合并单元，同时需提出一种更高效精确的人身触电电流识别方法，利用 STM32 单片机实现，并采用 Cat1.0 和高速电力线载波双模通信的方法，达到对整个路灯系统以及单个路灯的实时安全管控。

    • 主要职责：

      1. 设计并制作单相柔性智能配电合并单元；

      2. 建立人体阻抗的相关数学模型，用 Matlab 进行仿真分析，确定人体触电电流特征；

      3. 利用 C 语言在 STM32 单片机上完成接地故障识别、人身触电识别、触电保护、分闸合闸等程序的开发和调试；

      4. 在街道现场安装调试设备，验证可行性；

  • 单相柔性智能配电合并单元 公司新品研发（项目参与人） 2023.9—2024.1

    • 项目描述：城市箱变是路灯系统的配电环节，针对其目前存在电参量监测盲区、安全防护不全面、不能对电能质量进行有效调控的问题，本项目以“安全、节能”为理念开发一款以 disPIC30F6014A 单片机、电能计量芯片为核心，外扩继电器、电流互感器、显示屏，能够实现对城市箱变低压出线各回路的电参量监测、回路控制、故障保护及电能分配等功能，适应智慧城市建设的发展需求的单相智能配电合并单元。该产品已投入生产并成功销售。

    • 主要职责：

      1. 用 Altium Designer 完成各模块电路板的原理图设计和 PCB 板布局绘制；

      2. 使用 FameView 组态软件进行各功能的检测；

      3. 提交 BOM 材料清单，协调器件采购、样机焊接与硬件调试；

      4. 编写相关技术文档（产品使用说明书、技术及操作说明书、生产测试指导书等）。

  • 智慧路灯单灯控制器 公司新品研发（项目负责人） 2024.08—至今

    • 项目描述：针对目前路灯控制系统智能化程度不高、通讯链路复杂易受干扰导致电能浪费严重、运营维护不方便的问题，本项目以“简约、安全、节能”为理念开发一款以 stm32 单片机、电能计量芯片、电力线载波芯片为核心，外扩继电器、日历时钟的单灯控制器，能够实现多策略智能控制、柔性启动、测量保护等功能，适应智慧城市建设的发展需求。
